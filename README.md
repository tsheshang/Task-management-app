📝 To-Do List Application
Task management system with add/edit/delete features and live time display. Responsive design using HTML, CSS, and JavaScript. GitHub-hosted project.
🚀 Features

➕ Add tasks

⏰ Shows task creation time

✏️ Edit tasks

🗑️ Delete tasks

💾 Saves tasks using Local Storage

🛠️ Technologies

HTML

CSS

JavaScript

▶️ How to Run

Open index.html in your browser.

Add, edit, or delete tasks.

Tasks stay saved even after refresh.
